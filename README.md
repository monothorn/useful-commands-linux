# useful-commands-linux
Commands I like to use.
-----------------------------------------------------------------------------------------------------------------


1. List all available services on an ubuntu system
```
service --status-all

 [ + ]  acpid
 [ + ]  apparmor
 [ + ]  apport
 [ + ]  atd
 [ - ]  bootmisc.sh
 [ - ]  checkfs.sh
 [ - ]  checkroot-bootclean.sh
 [ - ]  checkroot.sh
 [ - ]  console-setup

```
2. List all remote branches for a repo

```
git status -r

origin/develop
origin/feature/f1
origin/feature/f2
```
